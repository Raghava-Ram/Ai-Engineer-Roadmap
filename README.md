# AI Engineer Roadmap

A comprehensive learning path and resource collection for aspiring AI Engineers.

## Features

- Structured learning path for AI Engineering
- Curated collection of free learning resources
- Interactive checklists to track progress
- Downloadable resources and PDFs
- Dark/light theme support

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn/pnpm
- Git

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ai-engineer-roadmap.git
   cd ai-engineer-roadmap
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   # or
   pnpm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/app` - Main application routes
  - `/roadmap` - Learning path and weekly breakdown
  - `/resources` - All learning resources in one place
  - `/checklists` - Progress tracking checklists
- `/components` - Reusable UI components
- `/lib` - Shared utilities and data
